Info Tech – Tools

Welcome to the Tools section! This page provides an overview of the information technologies and tools we've worked with throughout the course. Each tool listed below includes a brief explanation, highlights of its most useful features, and tips to help future students learn how to use them effectively. We'd love to hear your thoughts—please take a moment to fill out the feedback survey at the bottom of the page!

HTML5

HTML5 structures the content of your site. It defines headers, paragraphs, links, images, and interactive elements. I found semantic tags like <article> and <section> especially useful.
HTML5 Docs

CSS

CSS styles and arranges content visually. I liked using flexbox for alignment. Tip: Use external stylesheets for clean and reusable code.
CSS Docs

JavaScript

JavaScript adds interactivity. I used it to show and hide content based on radio button selection. Tip: Test your script in the browser console.
JavaScript Docs

Bootstrap

Bootstrap is a CSS framework for responsive design. Its navbar and card components saved me time. Tip: Use grid layout to avoid breakpoints.
Bootstrap

PHP

PHP handles form submission and connects to databases. I used it to send form data and process it safely. Tip: Always validate and sanitize inputs.
PHP.net

MySQL

MySQL stores form data. I used SQL commands to retrieve and analyze survey responses. Tip: Use phpMyAdmin for managing tables.
MySQL

XAMPP/MAMP

These tools simulate a server locally. They let me run PHP and MySQL offline. Tip: Restart Apache if changes don’t show up.
XAMPP | MAMP

GazeParser

An eye-tracking Python tool. I found its heatmap visualizations helpful for analyzing user focus. Tip: Check compatibility with your webcam before use.
GazeParser GitHub

DOM Manipulation

DOM methods let you modify page content on the fly. I used document.getElementById() and innerHTML a lot. Tip: Use browser dev tools to inspect the DOM.
DOM on MDN

Form Validation

Validation prevents bad data. I used required fields and JavaScript checks. Tip: Combine HTML5 and JS for best results.
Form Validation Guide
